Issue tracker

The Org community tracks issue on updates.orgmode.org - see this announcement.

You can subscribe to this RSS feed for new bugs.

If you want to report a bug, use M-x org-submit-bug-report or directly send an email to the Org mailing list.

Make sure to read Org’s manual instructions on how to provide useful feedback.

If you can reproduce a bug, that’s really helpful: reply to the original poster adding X-Bug-Woof: confirmed in your mail header.
